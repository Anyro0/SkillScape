# SkillScape
A machine learning (ML)-powered web application
##### Data from
https://www.kaggle.com/datasets/asaniczka/1-3m-linkedin-jobs-and-skills-2024/data

### How to run application
 - Download job_skills.csv and linkedin_job_postings.csv data files from the link provided above
 - Run app.py
 - go to http://localhost:5000/
